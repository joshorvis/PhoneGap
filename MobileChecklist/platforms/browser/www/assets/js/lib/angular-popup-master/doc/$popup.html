Class: $popup

$popup

new $popup($rootScope, $compile, $q, $timeout, $sce)

Popup module for angular applications This module has some cool UX functionality: - it will set focus on OK in simple alert popup - it will focus on input if you have one in the popup (will work in confirmation popup only)
